Project Description

This is a Food Delivery Web App built using React. It allows users to log in, browse restaurants, view menus, add items to cart, and place orders. The project focuses on building a complete frontend application with routing, state management, and a smooth user flow.

Features
User login (dummy authentication)
Browse restaurants on home page
View dynamic menu using URL parameters
Add items to cart
View and manage cart
Simple and responsive UI

Tech Stack
React
React Router
Context API
Tailwind CSS
